# 로또 번호 추첨기 프로젝트

## 1. 개요

이 프로젝트는 사용자에게 로또 번호를 추천해주는 웹 애플리케이션입니다. 사용자는 원하는 만큼의 로또 번호 조합을 생성하고 확인할 수 있습니다.

## 2. 기능

*   **번호 생성:** 1부터 45까지의 숫자 중 무작위로 6개를 선택하여 한 조합을 생성합니다.
*   **다중 조합 생성:** 사용자가 원하는 개수만큼의 로또 번호 조합을 한 번에 생성할 수 있습니다.
*   **결과 표시:** 생성된 번호 조합을 화면에 보기 쉽게 표시합니다.

## 3. 디자인

*   **언어:** 한국어
*   **레이아웃:** 사용자가 쉽게 이해하고 사용할 수 있는 직관적인 인터페이스를 제공합니다.
*   **스타일:** 깔끔하고 현대적인 디자인을 적용하여 사용자 경험을 향상시킵니다.

## 4. 현재 요청 계획

1.  **`index.html` 수정:**
    *   언어 설정을 `ko`로 변경합니다.
    *   페이지 제목을 "로또 번호 추첨기"로 변경합니다.
    *   사용자가 번호 조합 개수를 입력할 수 있는 입력 필드와 생성 버튼을 추가합니다.
    *   생성된 번호가 표시될 영역을 추가합니다.
2.  **`main.js` 수정:**
    *   로또 번호를 생성하는 로직을 구현합니다.
    *   사용자가 요청한 개수만큼 번호 조합을 생성하여 화면에 표시하는 기능을 구현합니다.
3.  **`style.css` 수정:**
    *   전체적인 레이아웃과 디자인을 개선하여 가독성과 시각적 매력을 높입니다.
